# Synapse AI API Specification

## Overview

The Synapse AI API provides access to debate orchestration, model interactions, consensus generation, evaluation metrics, and experiment management.

The API follows RESTful principles and uses JSON for request and response payloads.

Base URL:

```text
/api/v1
```

---

# Core Resources

The system revolves around four primary resources:

* Debates
* Responses
* Consensus Reports
* Evaluation Metrics

---

# Debate Lifecycle

```text
Create Debate
      │
      ▼
Generate Opinions
      │
      ▼

      ▼
Generate Consensus

      ▼

      ▼
```

# Debate Endpoints

## Create Debate

Creates a new debate session.

### Endpoint

```http
POST /debates
```

### Request

```json
{
  "question": "Should AI-generated code be deployed directly to production?",
  "models": [
    "Model-A",
    "Model-B",
    "Model-C"
  ]
}
```

### Response

```json
{
  "debate_id": "deb_12345",
  "status": "created",
  "created_at": "2026-06-18T12:00:00Z"
}
```

---

## Get Debate

Retrieve debate details.

### Endpoint

```http
GET /debates/{debate_id}
```

### Response

```json
{
  "debate_id": "deb_12345",
  "question": "...",
  "status": "completed",
  "created_at": "...",
  "completed_at": "..."
}
```

---

## List Debates

Retrieve previous debates.

### Endpoint

```http
GET /debates
```

### Query Parameters

```text
?page=1
&limit=20
```

### Response

```json
{
  "total": 24,
  "debates": []
}
```

# Debate Execution Endpoints

## Run Debate

Starts the complete debate pipeline.

This endpoint:

* Generates opinions
* Runs critiques
* Runs revisions
* Produces consensus
* Calculates metrics

### Endpoint

```http
POST /debates/{debate_id}/run
```

### Request

```json
{
  "rounds": 3
}
```

### Response

```json
{
  "debate_id": "deb_12345",
  "status": "running"
}
```

---

## Get Debate Status

Retrieve current execution state.

### Endpoint

```http
GET /debates/{debate_id}/status
```

### Response

```json
{
  "status": "completed",
  "current_phase": "evaluation",
  "progress": 100
}
```

# MVP API Scope

Version 1 includes:

* Debate creation
* Debate execution
* Opinion retrieval
* Transcript retrieval
* Consensus reports
* Evaluation metrics

Version 1 excludes:

* Authentication
* User accounts
* Team collaboration
* Webhooks
* Real-time streaming

---

# API Design Summary

The Synapse AI API is designed around debates as the primary resource. It exposes a clean lifecycle from debate creation through consensus generation and evaluation, enabling both interactive use cases and large-scale experimentation.
